## Problem

Given the head of a singly linked list, return true` if it is a palindrome or `false` otherwise`.

```md
Input: head = [1,2,2,1]
output: true
```



**Constraints:**

- The number of nodes in the list is in the range <code>[1, 10<sup>5</sup>]</code>.
- <code>0 &lt;= Node.val &lt;= 9</code>
